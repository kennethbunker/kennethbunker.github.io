CPLD

Chilean Political Landscape Dataset · 1989—2024

Home About Contact

Longitudinal dataset of district-level electoral outcomes covering three decades of Chilean democracy. Standardized measures of party system fragmentation, ideological orientation, and institutional change across legislative, regional, and municipal contests.

Code Examples

# Load CPLD dataset library(tidyverse) cpld <- read_csv("cpld_1989_2024.csv") # Calculate effective number of parties by year cpld %>% group_by(year) %>% summarise(mean_enpv = mean(enpv, na.rm = TRUE))
# Analyze fragmentation trends pre/post-reform cpld %>% mutate(period = if_else(year >= 2017, "Post-Reform", "Pre-Reform")) %>% ggplot(aes(x = year, y = enpv)) + geom_point(alpha = 0.3) + geom_smooth(method = "loess")
